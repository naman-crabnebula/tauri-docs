Send native notifications to your user using the notification plugin.

## Supported Platforms

- Windows
- macOS
- Linux
- iOS
- Android

## Setup

Install the notifications plugin to get started.

<Tabs>
    <TabItem label="Automatic">

        Use your project's package manager to add the dependency:

        <CommandTabs npm="npm run tauri add notification"
        yarn="yarn run tauri add notification"
        pnpm="pnpm tauri add notification"
        cargo="cargo tauri add notification" />

    </TabItem>
    <TabItem label="Manual">
        <Steps>

        1.  Run `cargo add tauri-plugin-notification` to add the plugin to the project's dependencies in `Cargo.toml`.

        2.  Modify `lib.rs` to initialize the plugin:

            ```rust ins={6}
            // lib.rs
            #[cfg_attr(mobile, tauri::mobile_entry_point)]
            pub fn run() {
                tauri::Builder::default()
                    // Initialize the plugin
                    .plugin(tauri_plugin_notification::init())
                    .run(tauri::generate_context!())
                    .expect("error while running tauri application");
            }
            ```

        3.  If you'd like to use notifications in JavaScript then install the npm package as well:

            <CommandTabs
                npm="npm install @tauri-apps/plugin-notification"
                yarn="yarn add @tauri-apps/plugin-notification"
                pnpm="pnpm add @tauri-apps/plugin-notification"
            />

        </Steps>
    </TabItem>

</Tabs>

### Requesting Permission

To send notifications to the user, appropriate permissions must be obtained beforehand.

<Steps>
<ol>
    <li>
    Check if permission is granted
    <Tabs>
        <TabItem label="Rust">
        ```rust {1} {6}
        use tauri_plugin_notification::NotificationExt;

        #[tauri::command]
        fn send_notification(app_handle: tauri::AppHandle) {
            // Do you have permission to send a notification?
            let mut state = app_handle.notification().permission_state().unwrap();
        }
        ```
        </TabItem>
        <TabItem label="JavaScript">
        ```js
        import isPermissionGranted from '@tauri-apps/plugin-notification';

        // Do you have permission to send a notification?
        let permissionGranted = await isPermissionGranted();
        ```
        </TabItem>
    </Tabs>
    </li>
    <li>
    Request permission if not granted
    <Tabs>
        <TabItem label="Rust">
        ```rust {2} {10-12}
        use tauri_plugin_notification::NotificationExt;
        use tauri_plugin_notification::PermissionState;

        #[tauri::command]
        fn send_notification(app_handle: tauri::AppHandle) {
            // Do you have permission to send a notification?
            let mut state = app_handle.notification().permission_state().unwrap();

            // If permissions are not already granted, we will need to request them.
            if state != PermissionState::Granted {
                state = app_handle.notification().request_permission().unwrap();
            }
        }
        ```
        </TabItem>
        <TabItem label="JavaScript">
        ```js {3} {10-13}
        import {
        isPermissionGranted,
        requestPermission,
        } from '@tauri-apps/plugin-notification';

        // Do you have permission to send a notification?
        let permissionGranted = await isPermissionGranted();

        // If permissions are not already granted, we will need to request them.
        if (!permissionGranted) {
        const permission = await requestPermission();
        permissionGranted = permission === 'granted';
        }
        ```
        </TabItem>
    </Tabs>
    </li>
</ol>
</Steps>

### Send Notification


<Tabs>
<TabItem label="Rust">

```rust ins={11-16}
use tauri_plugin_notification::NotificationExt;
use tauri_plugin_notification::PermissionState;

#[tauri::command]
fn send_notification(app_handle: tauri::AppHandle) {
    let mut state = app_handle.notification().permission_state().unwrap();

    // Once permission has been granted we can send the notification
    if state == PermissionState::Granted {
        // Build and send the Notification
        app_handle.notification()
            .builder()
            .title("Tauri")
            .body("Tauri is Awesome")
            .show()
            .unwrap();
    }
}
```

</TabItem>

<TabItem label="JavaScript">

```js ins={10}
import {
  isPermissionGranted,
  sendNotification,
} from '@tauri-apps/plugin-notification';

let permissionGranted = await isPermissionGranted();

// Once permission has been granted we can send the notification
if (permissionGranted) {
  sendNotification({ title: 'Tauri', body: 'Tauri is awesome!' });
}
```

</TabItem>
</Tabs>

### Actions

:::note[Supported Platforms]

This feature is exclusively available through **JavaScript** bindings and targets only **Android** devices for now.

:::

Notification actions are interactive elements within a notification that allow users to perform tasks directly from the notification itself, without opening the application. For example, sending the reply to messages, pausing the music etc.

To add Action to a notification, follow the given steps.

<Steps>
<ol>
    <li> 
    Add capabilities to the application

    ```json title=capabilities/default.json ins={3-4}
    {
        "permissions": [
                "notification:allow-register-action-types",
                "notification:allow-register-listener"
        ]
    }
    ```

### Channels

:::note[Supported Platforms]

This feature is exclusively targets only **Android** devices for now.

:::

Notification channels are a way to group and manage notifications in Android applications. By default, all the notifications are sent to `Default` channel. To send a notification in a custom channel, follow the below steps.

<Steps>
<ol>
    <li>
    Create a new channel builder with channel id and name and add the desired properties to it.

    <Tabs>
        <TabItem label="Rust">
        ```rust
        #[cfg(target_os = "android")]
        use tauri_plugin_notification::{Channel, Importance};

        fn notification_channel(app_handle: tauri::AppHandle) {
            #[cfg(target_os = "android")]{
                // Create a new channel builder with channel id and name
                let channel_builder = Channel::builder("tauri_channel_id", "Tauri Notification");

                // Create the channel from channel builder
                let channel = channel_builder

                    // Add some configurations to the channel and build it
                    .description("Describe the type of notifications channel contains")
                    .lights(true)
                    .vibration(false)
                    .importance(Importance::High)

                    .build();

                // Register the channel with the application
                app_handle.notification().create_channel(channel);
            }
        }
        ```
        </TabItem>
        <TabItem label="JavaScript">

        First of all, `create_channel` add capabilities to the application.

        ```json title=capabilities/default.json ins={3}
        {
            "permissions": [
                    "notification:allow-create-channel",
            ]
        }
        ```

        Then, add the following snippet in your JavaScript code to create a channel.

        ```js
        import {createChannel, Importance, Visibility} from "@tauri-apps/plugin-notification";

        await createChannel({
            id: 'tauri_channel_id',
            name: 'Tauri Notification',
            description: 'Describe the type of notifications channel contains',
            lights: true,
            vibration: false,
            importance: Importance.Default,
        });
        ```
        </TabItem>
    </Tabs>
    </li>
    <li> 
    Now, you can simply add the **channel id** to the notification that you want to send to this channel.

    <Tabs>
        <TabItem label="Rust">
        ```rust ins={7}
        fn send_notification(app_handle: tauri::AppHandle) {
            app_handle
                .notification()
                .builder()
                .title("Tauri")
                .body("Tauri channels are so easy to use in rust!")
                .channel_id("tauri_channel_id")
                .show()
                .expect("Failed to show the notification");
        }
        ```
        </TabItem>
        <TabItem label="JavaScript">

        ```js ins={4}
        sendNotification({
            title: 'Tauri',
            body: 'Tauri channels are so easy to use!',
            channelId: 'tauri_channel_id'
        });
        ```
        </TabItem>
    </Tabs>
    </li>
